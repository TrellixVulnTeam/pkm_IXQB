def uninstall(self, packages: _PACKAGE_NAMES_T) -> Set[str]

Description¶

attempt to remove the required packages from this env together will all the dependencies that may become orphan as a result of this step.

if a package p in packages is a dependency (directly or indirectly) of another “user requested” package q not in packages then p will be kept in the environment but its “user requested” flag will be removed (if it was existed)
