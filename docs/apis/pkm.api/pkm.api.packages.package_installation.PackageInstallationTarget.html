def uninstall(self, packages_to_remove: List[str]) -> Set[str]

Description¶

attempt to remove the required packages from this target together will all the dependencies that may become orphan as a result of this step.

if a package p in packages is a dependency (directly or indirectly) of another “user requested” package q not in packages then p will be kept in the target but its “user requested” flag will be removed (if it was existed)

def force_remove(self, package: str)

Description¶

def plan_installation(self, dependencies: List[Dependency],
    repository: Repository, user_requested: bool = True,
    dependencies_override: Optional[Dict[str, Dependency]] = None,
    updates: Optional[List[str]] = None) -> InstallationPlan

Description¶

plan but does not install an installation for the given dependencies. resolve the dependencies using the given repository, making sure to not break any pre-installed “user-requested” packages (but may upgrade their dependencies if it needs to)
