renders the given template into the target directory

Parameters

  • template_path – the directory/resource holding the template

  • target_dir – the directory to output the generated content into

  • args – positional arguments for the template

  • kwargs – named arguments for the template

  • extra_context – extra variables that will be available inside proto.py

  • excluded_files – list of path objects that represents files in the template directory that should be excluded from the generation process

  • allow_overwrite – if True, files that are already exists will be overridden by the template
