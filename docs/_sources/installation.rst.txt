Installation
============

System Requirements
-------------------
``pkm`` requires ``python 3.8+`` and can run on any supported OS.

How to Install
--------------
There are two main ways to install pkm on your system:

1. if you already have another package manager installed (most probably :program:`pip`)

.. code-block:: console

    $ pip install pkm-cli


2. Alternatively, you can download :download:`install-pkm.py </downloads/install-pkm-cli.py>` and execute

.. code-block:: console

    $ python install-pkm-cli.py


.. note::
     pkm-cli is distributed as a containerized application package,
     it built in a way such that there can be no conflicts with other packages installed in
     your system or virtual environment
